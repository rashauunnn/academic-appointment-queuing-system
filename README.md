# Academic Appointment & Queuing System
A PHP-based system designed to streamline student-faculty appointments with real-time tracking.

### Key Features:
- **Student Dashboard:** Real-time queue tracking and appointment history.
- **Faculty Management:** Availability toggles (Available/Break/Out) and queue handling.
- **Admin Analytics:** Peak hours identification and traffic monitoring.
- **Security:** Role-based access control (RBAC).

### Technologies Used:
- PHP (PDO)
- MySQL
- Tailwind CSS
- JavaScript (Fetch API/Polling)
